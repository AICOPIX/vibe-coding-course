3.4 GitHub 仓库、上传、协作和发布准备

GitHub 不只是远程备份，它也是项目协作入口和后续发布流程的重要连接点。

已整理 核心课

本节目标

把仓库托管、协作和发布准备连成一条线。

核心要点

  • 仓库是项目事实来源，不只是代码抽屉。
  • 页面目录、课程目录和工作流都应该进入仓库。
  • 对个人项目来说，GitHub 也是最自然的发布备份中心。

本节练习

解释为什么“发布配置也要放进仓库”比口头说明更可靠。
