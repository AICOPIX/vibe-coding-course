# AGENTS

## 项目定位

这是一个面向个人 AI 编程学习的课程仓库。

- `课程/` 存放 Markdown 课程原文。
- `网页/` 存放 GitHub Pages 直接发布的静态页面。
- `素材/` 和 `.shared/` 仅作为参考资源，不是规范来源。

## Codex 协作规则

- 先读取当前结构，再修改文件。
- 课程编号、Markdown 文件名、HTML 文件名必须一致。
- `课程/课程目录.md`、`网页/index.html` 和实际文件数量必须同步。
- 只使用相对路径，不依赖本地绝对路径。
- 不新增隐藏规则目录。

## 命名规则

- 阶段目录格式：`第X阶段-名称`
- 课程文件格式：`X.Y-课程标题.md`
- 页面文件格式：`X.Y-课程标题.html`

## 页面同步规则

- 每一节核心课程都必须有对应 HTML 页面。
- 页面必须包含返回目录、上一节、下一节导航。
- 如果课程内容尚未扩展完整，也必须生成可访问的占位页面。

## GitHub Pages 约束

- 发布源固定为 `网页/`
- 首页固定为 `网页/index.html`
- 样式共享文件固定为 `网页/css/style.css`
