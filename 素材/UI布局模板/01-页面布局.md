# 📐 页面布局模板

> 复制你需要的布局，粘贴给 AI 使用

---

## 1. 单栏居中布局

**适用场景**：博客文章、落地页、简单展示页

### ASCII 结构图
```
┌─────────────────────────────────────────────────┐
│                    Header                       │  高度: 64px
├─────────────────────────────────────────────────┤
│                                                 │
│              ┌─────────────────┐                │
│              │                 │                │
│              │     Content     │                │  最大宽度: 800px
│              │     (居中)       │                │  左右自动留白
│              │                 │                │
│              └─────────────────┘                │
│                                                 │
├─────────────────────────────────────────────────┤
│                    Footer                       │  高度: 80px
└─────────────────────────────────────────────────┘
```

### Tailwind 代码
```html
<div class="min-h-screen flex flex-col">
  <!-- Header -->
  <header class="h-16 border-b flex items-center px-6">
    <span>Logo</span>
  </header>
  
  <!-- Content -->
  <main class="flex-1 py-12">
    <div class="max-w-3xl mx-auto px-6">
      <!-- 你的内容 -->
    </div>
  </main>
  
  <!-- Footer -->
  <footer class="h-20 border-t flex items-center justify-center">
    <span>© 2025</span>
  </footer>
</div>
```

## 2. 左侧边栏布局

**适用场景**：后台管理、文档站点、设置页面


### ASCII 结构图
```
┌─────────────────────────────────────────────────┐
│                    Header                       │  高度: 64px
├──────────┬──────────────────────────────────────┤
│          │                                      │
│          │                                      │
│ Sidebar  │            Content                   │
│ (固定)    │            (自适应)                  │
│          │                                      │
│  240px   │            flex: 1                   │
│          │                                      │
│          │                                      │
└──────────┴──────────────────────────────────────┘
```

### Tailwind 代码
```html
<div class="min-h-screen flex flex-col">
  <!-- Header -->
  <header class="h-16 border-b flex items-center px-6">
    <span>Logo</span>
  </header>
  
  <!-- Body -->
  <div class="flex flex-1">
    <!-- Sidebar -->
    <aside class="w-60 border-r p-4">
      <nav>
        <a href="#" class="block py-2 px-4 rounded hover:bg-gray-100">菜单1</a>
        <a href="#" class="block py-2 px-4 rounded hover:bg-gray-100">菜单2</a>
        <a href="#" class="block py-2 px-4 rounded hover:bg-gray-100">菜单3</a>
      </nav>
    </aside>
    
    <!-- Content -->
    <main class="flex-1 p-6">
      <!-- 你的内容 -->
    </main>
  </div>
</div>
```

## 3. 右侧边栏布局

**适用场景**：博客（侧边栏放目录）、电商（侧边栏放筛选）

### ASCII 结构图
```
┌─────────────────────────────────────────────────┐
│                    Header                       │  高度: 64px
├──────────────────────────────────┬──────────────┤
│                                  │              │
│                                  │              │
│            Content               │   Sidebar    │
│            (自适应)               │   (固定)     │
│                                  │              │
│            flex: 1               │    280px     │
│                                  │              │
│                                  │              │
└──────────────────────────────────┴──────────────┘
```

### Tailwind 代码
```html
<div class="min-h-screen flex flex-col">
  <!-- Header -->
  <header class="h-16 border-b flex items-center px-6">
    <span>Logo</span>
  </header>
  
  <!-- Body -->
  <div class="flex flex-1">
    <!-- Content -->
    <main class="flex-1 p-6">
      <!-- 你的内容 -->
    </main>
    
    <!-- Sidebar -->
    <aside class="w-70 border-l p-4">
      <div class="sticky top-4">
        <h3 class="font-bold mb-4">目录</h3>
        <!-- 侧边栏内容 -->
      </div>
    </aside>
  </div>
</div>
```

## 4. 三栏布局

### ASCII 结构图
```
┌─────────────────────────────────────────────────┐
│                    Header                       │  高度: 64px
├──────────┬───────────────────────┬──────────────┤
│          │                       │              │
│          │                       │              │
│  Left    │       Content         │    Right     │
│ Sidebar  │       (自适应)         │   Sidebar    │
│          │                       │              │
│  200px   │       flex: 1         │    240px     │
│          │                       │              │
│          │                       │              │

```

### Tailwind 代码
```html
<div class="min-h-screen flex flex-col">

    <span>Logo</span>
  </header>
  
  <!-- Body -->

    <!-- Left Sidebar -->
    <aside class="w-50 border-r p-4">
      <nav>导航菜单</nav>
    </aside>
    
    <!-- Content -->
    <main class="flex-1 p-6">
      <!-- 你的内容 -->
    </main>
    
    <!-- Right Sidebar -->
    <aside class="w-60 border-l p-4">

  </div>
</div>
```

## 5. Dashboard 布局（可折叠侧边栏）

**适用场景**：管理后台、数据面板

### ASCII 结构图
```
┌──────────┬──────────────────────────────────────┐
│          │              Header                  │  高度: 64px
│          ├──────────────────────────────────────┤
│          │                                      │
│ Sidebar  │                                      │
│ (可折叠)  │            Content                   │
│          │            (带内边距)                 │
│  64px    │                                      │
│   或     │            padding: 24px             │
│  240px   │                                      │
│          │                                      │
└──────────┴──────────────────────────────────────┘
```

### Tailwind 代码
```html
<div class="min-h-screen flex">
  <!-- Sidebar -->
  <aside class="w-60 bg-gray-900 text-white flex flex-col">
    <div class="h-16 flex items-center px-4 border-b border-gray-700">
      <span class="font-bold">Dashboard</span>
    </div>
    <nav class="flex-1 p-4">
      <a href="#" class="block py-2 px-4 rounded hover:bg-gray-800">首页</a>
      <a href="#" class="block py-2 px-4 rounded hover:bg-gray-800">数据</a>
      <a href="#" class="block py-2 px-4 rounded hover:bg-gray-800">设置</a>
    </nav>
  </aside>
  
  <!-- Main -->
  <div class="flex-1 flex flex-col">
    <!-- Header -->
    <header class="h-16 border-b flex items-center justify-between px-6">
      <span>页面标题</span>
      <div>用户头像</div>
    </header>
    
    <!-- Content -->
    <main class="flex-1 p-6 bg-gray-50">
      <!-- 你的内容 -->
    </main>
  </div>
</div>
```
