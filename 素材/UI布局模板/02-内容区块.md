## 1. Hero 居中版

**适用场景**：落地页首屏、产品介绍

### ASCII 结构图
```
┌─────────────────────────────────────────────────┐
│                                                 │
│                                                 │
│              大标题（48px，居中）                 │
│                                                 │
│           副标题/描述（18px，居中）               │  高度: 500px
│                                                 │
│            [主按钮]    [次按钮]                  │
│                                                 │
│                                                 │
└─────────────────────────────────────────────────┘
```

### Tailwind 代码
```html
<section class="h-[500px] flex flex-col items-center justify-center text-center px-6">
  <h1 class="text-5xl font-bold mb-4">你的产品名称</h1>
  <p class="text-xl text-gray-600 mb-8 max-w-2xl">
    一句话描述你的产品价值，让用户立刻明白这是什么
  </p>
  <div class="flex gap-4">
    <button class="px-6 py-3 bg-blue-500 text-white rounded-lg hover:bg-blue-600">
      立即开始
    </button>
    <button class="px-6 py-3 border border-gray-300 rounded-lg hover:bg-gray-50">
      了解更多
    </button>
  </div>
</section>
```

## 2. Hero 左右分栏版

**适用场景**：产品展示、功能介绍

### ASCII 结构图
```
┌─────────────────────────────────────────────────┐
│                                                 │
│   ┌─────────────────┐  ┌─────────────────────┐  │
│   │                 │  │                     │  │
│   │    文字区域      │  │      图片区域        │  │
│   │                 │  │                     │  │  高度: 500px
│   │  标题           │  │      [图片]         │  │
│   │  描述           │  │                     │  │
│   │  [按钮]         │  │                     │  │
│   │                 │  │                     │  │
│   └─────────────────┘  └─────────────────────┘  │
│        50%                    50%               │
└─────────────────────────────────────────────────┘
```

### Tailwind 代码
```html
<section class="py-20">
  <div class="max-w-6xl mx-auto px-6 flex items-center gap-12">
    <!-- 文字区域 -->
    <div class="flex-1">
      <h1 class="text-4xl font-bold mb-4">产品标题</h1>
      <p class="text-lg text-gray-600 mb-6">
        详细描述你的产品功能和价值，让用户了解为什么要选择你
      </p>
      <button class="px-6 py-3 bg-blue-500 text-white rounded-lg">
        立即体验
      </button>
    </div>
    <!-- 图片区域 -->
    <div class="flex-1">
      <img src="your-image.png" alt="产品图" class="rounded-lg shadow-lg">
    </div>
  </div>
</section>
```

## 4. 图文左右布局（交替）

**适用场景**：功能详情、产品介绍

### ASCII 结构图
```
┌─────────────────────────────────────────────────┐
│   ┌─────────────────┐  ┌─────────────────────┐  │
│   │     [图片]       │  │  标题               │  │
│   │                 │  │  描述文字            │  │  第1行
│   │                 │  │  [按钮]              │  │
│   └─────────────────┘  └─────────────────────┘  │
├─────────────────────────────────────────────────┤
│   ┌─────────────────┐  ┌─────────────────────┐  │
│   │  标题           │  │     [图片]           │  │
│   │  描述文字        │  │                     │  │  第2行（反向）
│   │  [按钮]         │  │                     │  │
│   └─────────────────┘  └─────────────────────┘  │
└─────────────────────────────────────────────────┘
```

### Tailwind 代码
```html
<!-- 第1行：图左文右 -->
<section class="py-16">
  <div class="max-w-6xl mx-auto px-6 flex items-center gap-12">
    <div class="flex-1">
      <img src="feature1.png" alt="" class="rounded-lg">
    </div>

<!-- 第2行：文左图右（flex-row-reverse） -->
<section class="py-16 bg-gray-50">

      <img src="feature2.png" alt="" class="rounded-lg">

</section>
```

## 6. 微信公众号风格（居中窄版）

**适用场景**：文章阅读、内容展示

### ASCII 结构图
```
┌─────────────────────────────────────────────────┐
│                                                 │
│              ┌─────────────────┐                │
│              │                 │                │
│              │    内容区域      │                │
│              │                 │                │
│              │  最大宽度 680px  │                │  两边自动留白
│              │  左右 padding   │                │
│              │                 │                │
│              │  段落间距 24px   │                │
│              │                 │                │
│              └─────────────────┘                │
│                                                 │
└─────────────────────────────────────────────────┘
```

### Tailwind 代码
```html
<article class="py-12">
  <div class="max-w-[680px] mx-auto px-6">
    <!-- 标题 -->
    <h1 class="text-3xl font-bold mb-4">文章标题</h1>
    <p class="text-gray-500 mb-8">2025-01-08 · 作者名</p>
    
    <!-- 正文 -->
    <div class="prose prose-lg">
      <p>第一段内容...</p>
      <p>第二段内容...</p>
      <h2>小标题</h2>
      <p>更多内容...</p>
    </div>
  </div>
</article>
```
