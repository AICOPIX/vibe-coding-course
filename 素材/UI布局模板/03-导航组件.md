# 🧭 导航组件模板

> 复制你需要的组件，粘贴给 AI 使用

---

## 1. 简洁 Header

**适用场景**：简单网站、博客

### ASCII 结构图
```
┌─────────────────────────────────────────────────┐
│  Logo          首页   关于   作品   联系         │  高度: 64px
└─────────────────────────────────────────────────┘
     ↑                    ↑
   左对齐              右对齐
```

### Tailwind 代码
```html
<header class="h-16 border-b">
  <div class="max-w-6xl mx-auto px-6 h-full flex items-center justify-between">
    <!-- Logo -->
    <a href="/" class="text-xl font-bold">Logo</a>
    
    <!-- 导航 -->
    <nav class="flex gap-6">
      <a href="#" class="text-gray-600 hover:text-gray-900">首页</a>
      <a href="#" class="text-gray-600 hover:text-gray-900">关于</a>
      <a href="#" class="text-gray-600 hover:text-gray-900">作品</a>
      <a href="#" class="text-gray-600 hover:text-gray-900">联系</a>
    </nav>
  </div>
</header>
```

## 2. 带按钮 Header

**适用场景**：SaaS 产品、需要登录的网站

### ASCII 结构图
```
┌─────────────────────────────────────────────────┐
│  Logo      首页  功能  价格  文档    [登录] [注册]│  高度: 64px
└─────────────────────────────────────────────────┘
     ↑              ↑                    ↑
   左对齐         居中               右对齐
```

### Tailwind 代码
```html
<header class="h-16 border-b">
  <div class="max-w-6xl mx-auto px-6 h-full flex items-center justify-between">
    <!-- Logo -->
    <a href="/" class="text-xl font-bold">Logo</a>
    
    <!-- 中间导航 -->
    <nav class="flex gap-6">
      <a href="#" class="text-gray-600 hover:text-gray-900">首页</a>
      <a href="#" class="text-gray-600 hover:text-gray-900">功能</a>
      <a href="#" class="text-gray-600 hover:text-gray-900">价格</a>
      <a href="#" class="text-gray-600 hover:text-gray-900">文档</a>
    </nav>
    
    <!-- 右侧按钮 -->
    <div class="flex gap-3">
      <button class="px-4 py-2 text-gray-600 hover:text-gray-900">登录</button>
      <button class="px-4 py-2 bg-blue-500 text-white rounded-lg hover:bg-blue-600">
        注册
      </button>
    </div>
  </div>
</header>
```

## 3. 固定侧边栏

**适用场景**：后台管理、文档站点


### ASCII 结构图
```
┌──────────────┐
│    Logo      │  高度: 64px
├──────────────┤
│              │
│  📊 仪表盘   │
│  📁 项目     │
│  👥 用户     │
│  ⚙️ 设置    │
│              │
│              │
├──────────────┤
│  👤 用户名   │
│  退出登录    │
└──────────────┘
   宽度: 240px
```

### Tailwind 代码
```html
<aside class="w-60 h-screen bg-gray-900 text-white flex flex-col fixed left-0 top-0">
  <!-- Logo -->
  <div class="h-16 flex items-center px-6 border-b border-gray-700">
    <span class="text-xl font-bold">Logo</span>
  </div>
  
  <!-- 导航菜单 -->
  <nav class="flex-1 p-4">
    <a href="#" class="flex items-center gap-3 px-4 py-3 rounded-lg bg-gray-800">
      <span>📊</span> 仪表盘
    </a>
    <a href="#" class="flex items-center gap-3 px-4 py-3 rounded-lg hover:bg-gray-800">
      <span>📁</span> 项目
    </a>
    <a href="#" class="flex items-center gap-3 px-4 py-3 rounded-lg hover:bg-gray-800">
      <span>👥</span> 用户
    </a>
    <a href="#" class="flex items-center gap-3 px-4 py-3 rounded-lg hover:bg-gray-800">
      <span>⚙️</span> 设置
    </a>
  </nav>
  
  <!-- 底部用户信息 -->
  <div class="p-4 border-t border-gray-700">
    <div class="flex items-center gap-3 px-4 py-2">
      <div class="w-8 h-8 bg-gray-600 rounded-full"></div>
      <div>
        <div class="text-sm font-medium">用户名</div>
        <div class="text-xs text-gray-400">退出登录</div>
      </div>
    </div>
  </div>
</aside>
```

## 5. 多栏 Footer

**适用场景**：企业网站、产品官网

### ASCII 结构图
```
┌─────────────────────────────────────────────────┐
│                                                 │
│   Logo          产品        资源        关于     │
│   简介          功能1       文档        团队     │
│                 功能2       博客        联系     │
│                 价格        帮助        招聘     │
│                                                 │
├─────────────────────────────────────────────────┤
│   © 2025 公司名              隐私政策 | 服务条款  │
└─────────────────────────────────────────────────┘
```

```html
<footer class="bg-gray-900 text-white">

      <div>
        <h4 class="font-bold mb-4">产品</h4>

          <li><a href="#" class="hover:text-white">功能</a></li>

      </div>
```
