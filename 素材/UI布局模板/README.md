# 📐 UI 布局模板库

这是一个可复用的布局模板集合，用于 AI 编程时快速描述页面结构。

## 使用方法

1. 找到你需要的布局模板
2. 复制 ASCII 结构图和代码
3. 粘贴给 AI，告诉它"按照这个布局来做"
4. 根据需要修改具体内容

## 模板分类

| 文件 | 内容 |
|------|------|
| `01-页面布局.md` | 单栏、双栏、三栏、Dashboard |
| `02-内容区块.md` | Hero、卡片网格、图文组合 |
| `03-导航组件.md` | Header、Sidebar、Footer |
| `04-表单布局.md` | 登录、注册、设置页面 |
| `05-移动端布局.md` | 手机端常用布局 |
