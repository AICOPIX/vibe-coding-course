# Vibe Coding Course

这是一个面向个人 AI 编程学习的课程仓库，包含两层内容：

- `课程/`：课程 Markdown 原文
- `网页/`：可直接部署到 GitHub Pages 的静态课程站

## 本地预览

直接在浏览器打开 [index.html](C:/AI/vibe-coding-course/网页/index.html) 即可查看站点首页。

课程已经整理为 8 个阶段、35 节核心课，重点压缩了重复定义课，保留了实战主线。
