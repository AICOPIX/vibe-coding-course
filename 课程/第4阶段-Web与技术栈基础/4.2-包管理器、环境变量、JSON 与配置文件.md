# 4.2 包管理器、环境变量、JSON 与配置文件

> 学习状态：已整理
> 所属阶段：第4阶段 Web 与技术栈基础
> 内容说明：合并旧 4.1、4.2、4.3、4.7。

## 本节目标

理解项目运行所需的几类基础配置。

## 课程摘要

这些概念经常一起出现，本质都在解决“项目如何被配置和运行”。

## 核心要点

- 包管理器管理依赖，JSON 常用于保存结构化配置。
- 环境变量适合存放会因环境变化的参数。
- 配置文件的价值在于让项目规则可重复、可共享。

## 本节练习

说出为什么 API 密钥不应该直接写进页面文件。

## 学完后你应该能做到

- 用自己的话解释本节核心概念
- 把本节内容放回当前项目里判断它的实际作用
- 在后续页面或项目中继续复用这里的方法
