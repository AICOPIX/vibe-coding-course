# 4.3 TypeScript、SDK 与版本号

> 学习状态：已整理
> 所属阶段：第4阶段 Web 与技术栈基础
> 内容说明：合并旧 4.4、4.5、4.6。

## 本节目标

认识现代项目里常见的开发术语。

## 课程摘要

这节课回答三个高频问题：为什么有人用 TypeScript、SDK 到底是什么、版本号为什么会影响项目稳定性。

## 核心要点

- TypeScript 通过类型信息帮助你更早发现问题。
- SDK 是某项服务或平台给开发者准备的接入工具包。
- 版本号变化会影响功能兼容性，升级前要看清破坏性改动。

## 本节练习

分别用一句通俗的话解释 TypeScript、SDK 和版本号。

## 学完后你应该能做到

- 用自己的话解释本节核心概念
- 把本节内容放回当前项目里判断它的实际作用
- 在后续页面或项目中继续复用这里的方法
