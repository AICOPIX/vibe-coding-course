# 4.4 前端框架选择：React、Vue、Next.js

> 学习状态：已整理
> 所属阶段：第4阶段 Web 与技术栈基础
> 内容说明：合并旧 4.8、4.9、4.10。

## 本节目标

建立框架选择的比较视角。

## 课程摘要

框架不是越新越好，而是看它是否适合团队、项目和部署方式。

## 核心要点

- React 更像灵活工具箱，Vue 更强调上手友好。
- Next.js 是基于 React 的完整框架，适合更复杂的站点和应用。
- 个人静态课程站不一定需要框架，简单 HTML 也可能是更稳的方案。

## 本节练习

说明为什么当前课程站保留纯静态 HTML，而不是立即改成框架项目。

## 学完后你应该能做到

- 用自己的话解释本节核心概念
- 把本节内容放回当前项目里判断它的实际作用
- 在后续页面或项目中继续复用这里的方法
