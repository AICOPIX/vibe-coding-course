# 4.1 网页的组成与 HTML、CSS、JavaScript 分工

> 学习状态：已整理
> 所属阶段：第4阶段 Web 与技术栈基础
> 内容说明：合并旧 6.1 到 6.4。

## 本节目标

从一个页面的真实结构理解前端三件套。

## 课程摘要

不再把网页基础拆成四节定义课，而是直接用一个完整页面理解结构、样式和交互的分工。

## 核心要点

- HTML 负责结构和语义。
- CSS 负责视觉、布局和响应式。
- JavaScript 负责状态切换、交互行为和数据更新。

## 本节练习

用当前课程站举例，分别指出 HTML、CSS、JavaScript 各做了什么。

## 学完后你应该能做到

- 用自己的话解释本节核心概念
- 把本节内容放回当前项目里判断它的实际作用
- 在后续页面或项目中继续复用这里的方法
