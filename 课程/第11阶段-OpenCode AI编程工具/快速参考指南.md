## ⚡ 快速开始（5分钟）

### 1. 安装

```bash
# 方式 1：安装脚本（推荐）
curl -fsSL https://opencode.ai/install.sh | sh

# 方式 2：Homebrew
brew install opencode

# 方式 3：下载桌面应用
# 访问 https://opencode.ai/download
```

### 2. 配置

```bash
# 启动 OpenCode
opencode

# 在 OpenCode 中运行
/connect

# 选择 opencode，访问 https://opencode.ai/auth
# 复制 API 密钥，粘贴到 OpenCode
```

### 3. 初始化项目

```bash
cd /path/to/your/project
opencode init
```

### 4. 开始使用

```bash
opencode

# 在提示符后输入你的需求
> 请解释这个项目的结构
```

## 💡 提示

### 提示 1：给出清晰的需求

❌ 不好：`> 创建一个组件`

✅ 好：`> 创建一个用户列表组件，显示用户名、邮箱和删除按钮，使用 React 和 Tailwind CSS`

### 提示 2：使用 Plan 模式

对于复杂功能，先用 Plan 模式查看计划，然后再构建。

### 提示 3：分步骤进行

大项目分成多个小任务，逐个完成。

### 提示 4：定期保存

使用 Git 保存每个阶段的代码。

### 提示 5：充分测试

每完成一个功能就测试一次。

## 🚨 常见错误

### 错误 1：需求太模糊

```
❌ > 创建一个网站
✅ > 创建一个个人作品集网站，包含首页、关于页面、作品展示和联系方式
```

### 错误 2：一次要求太多

```
❌ > 创建整个电商系统
✅ > 第一步：创建产品列表页面
   > 第二步：添加购物车功能
   > 第三步：添加支付功能
```

### 错误 3：不检查结果

```
❌ 直接使用 OpenCode 生成的代码
✅ 检查代码，测试功能，然后再使用
```

## 🔐 安全建议

1. **不要在公开代码中暴露 API 密钥**
   ```bash
   # 使用环境变量
   export ANTHROPIC_API_KEY="your-key"
   ```

2. **定期更新 OpenCode**
   ```bash
   # 启用自动更新
   # 在配置文件中设置 "autoupdate": true
   ```

3. **备份你的配置**
   ```bash
   cp ~/.config/opencode/opencode.json ~/backup/
   ```

4. **使用 .gitignore**
   ```
   # 不要提交 API 密钥
   .env
   .env.local
   opencode.json
   ```
