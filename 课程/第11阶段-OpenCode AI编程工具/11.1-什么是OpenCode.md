## 📌 核心概念

**OpenCode** 是一个**开源 AI 编程代理**。简单来说，它就像一个住在你电脑里的"AI 编程助手"，可以帮你：

- 📝 理解代码
- ✨ 添加新功能
- 🐛 修复 Bug
- 💡 给出编程建议

## 🎯 OpenCode 能做什么

### 1. 问代码问题
你可以问 OpenCode 关于你项目的任何问题，它会帮你解释代码。

**实际例子**：

```
你：这个 auth.js 文件是干什么的？

OpenCode：
这个文件处理用户身份验证。它包含：
1. validateToken() - 检查用户的登录令牌是否有效
2. generateToken() - 为新用户创建登录令牌
3. refreshToken() - 更新过期的令牌

这些函数保护你的应用，确保只有登录用户才能访问某些功能。
```

### 2. 添加新功能
告诉 OpenCode 你想要什么功能，它会帮你写代码。

**实际例子**：

```
你：给我的网站添加一个"关于我们"页面。
需要包含：
- 公司简介（2-3段文字）
- 团队成员卡片（3个成员）
- 联系方式

OpenCode：
我会为你创建：
1. about.html - 新页面
2. 添加到导航栏的链接
3. 使用 Tailwind CSS 美化样式
4. 响应式设计（手机和电脑都能看）

[OpenCode 生成代码...]
```

### 3. 制定计划（Plan 模式）
在真正写代码之前，OpenCode 可以先给你一个详细的计划。

**实际例子**：

```
你：我想给应用添加用户登录功能

OpenCode（Plan 模式）：
我会这样实现：

第一步：创建登录表单
- 用户名输入框
- 密码输入框
- 登录按钮

第二步：后端验证
- 检查用户名和密码
- 生成登录令牌
- 保存到数据库

第三步：前端处理
- 登录成功 → 跳转到首页
- 登录失败 → 显示错误信息

你觉得这个方案怎么样？
```

### 4. 修改和撤销
如果 OpenCode 做的不对，你可以轻松修改。

**实际例子**：

```
你：创建一个按钮

OpenCode：[创建了一个蓝色按钮]

你：不对，我想要红色的，并且更大

OpenCode：[修改为红色大按钮]

你：还是不对，用 /undo 撤销

OpenCode：[恢复到之前的状态]

你：重新做，这次用绿色

OpenCode：[创建绿色按钮]
```

## �️ OpenCode 的三种使用方式

### 方式 1：终端（TUI）
在命令行里使用 OpenCode，像聊天一样交互。

**优点**：
- 快速
- 不需要打开其他软件
- 适合开发者

**启动方式**：
```bash
opencode
```

### 方式 2：桌面应用
下载 OpenCode 的桌面版本，像使用普通软件一样。

**优点**：
- 界面友好
- 不需要懂命令行
- 适合初学者

**下载地址**：https://opencode.ai/download

**支持系统**：
- macOS (Apple Silicon)
- macOS (Intel)
- Windows
- Linux (.deb, .rpm, AppImage)

### 方式 3：IDE 扩展
在你的代码编辑器（如 VS Code）里直接使用 OpenCode。

**优点**：
- 不用切换窗口
- 集成度高
- 工作流顺畅

## 🤖 OpenCode 的两个工作模式

### 模式 1：Build（构建模式）
OpenCode 会直接修改你的代码。

**适合**：
- 你已经确定要做什么
- 你信任 OpenCode 的方案

**切换方式**：按 `Tab` 键

### 模式 2：Plan（计划模式）
OpenCode 只给出计划，不修改代码。

**适合**：
- 你想先看看方案
- 你想给出反馈
- 你想确保方向正确

**切换方式**：按 `Tab` 键

## 💡 实际使用场景

### 场景 1：快速理解陌生代码

```
你：我接手了一个旧项目，不知道从哪里开始。
请分析 src/ 目录的结构，告诉我各个文件的作用。

OpenCode：
项目结构分析：

src/
├── components/     - React 组件
│   ├── Header.jsx  - 页面顶部导航
│   ├── Footer.jsx  - 页面底部
│   └── Card.jsx    - 卡片组件
├── pages/          - 页面文件
│   ├── Home.jsx    - 首页
│   └── About.jsx   - 关于页面
├── utils/          - 工具函数
│   ├── api.js      - API 调用
│   └── helpers.js  - 辅助函数
└── styles/         - 样式文件

这是一个 React 应用，主要用于展示内容。
```
