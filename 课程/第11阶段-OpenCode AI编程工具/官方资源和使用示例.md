## 🎯 实际使用示例

### 示例 1：快速理解项目

**场景**：你接手了一个陌生的项目

**命令**：
```
> 分析这个项目的结构和主要功能
```

**OpenCode 回答**：
```
这是一个 React + Node.js 的电商应用。

前端（React）：
- src/components/ - 可复用组件
- src/pages/ - 页面
- src/hooks/ - 自定义 Hooks
- src/services/ - API 调用

后端（Node.js）：
- routes/ - API 路由
- controllers/ - 业务逻辑
- models/ - 数据模型
- middleware/ - 中间件

主要功能：
1. 产品浏览和搜索
2. 购物车管理
3. 用户认证
4. 订单管理
5. 支付处理
```

### 示例 2：添加新功能

**场景**：需要添加用户评论功能

**命令**：
```
> 给产品详情页添加评论功能
> 需要：
> - 评论表单（用户名、评分、评论内容）
> - 评论列表（显示所有评论）
> - 评论排序（最新优先）
> - 删除自己的评论
> 使用 React 和 Node.js
```

**OpenCode 生成**：
```
✓ 创建了 CommentForm.jsx 组件
✓ 创建了 CommentList.jsx 组件
✓ 添加了后端 API 端点
✓ 集成了数据库存储
✓ 添加了权限验证

完成！评论功能已添加。
```

### 示例 3：修复 Bug

**场景**：用户反馈登录后页面卡住

**命令**：
```
> 登录功能有问题。用户输入正确的用户名和密码后，
> 页面卡住了，没有跳转到首页。
> 请检查并修复这个问题。
```

**OpenCode 分析**：
```
我找到问题了！

在 login.js 第 67 行，缺少错误处理。
当登录成功时，应该调用 redirect() 函数，
但代码中没有这一步。

修复方案：
1. 添加登录成功的回调
2. 清除加载状态
3. 重定向到首页

修复中...
```

### 示例 4：性能优化

**场景**：网站加载很慢

**命令**：
```
> 网站加载很慢，特别是首页。
> 请分析性能问题并给出优化建议。
```

**OpenCode 建议**：
```
我分析了你的代码，发现以下性能问题：

1. 图片没有优化
   - 建议：使用 WebP 格式，压缩大小
   - 预期改进：减少 60% 的图片大小

2. 没有代码分割
   - 建议：使用 React.lazy() 分割代码
   - 预期改进：首屏加载时间减少 40%

3. 没有缓存策略
   - 建议：添加浏览器缓存和 CDN
   - 预期改进：重复访问速度提升 80%

4. 数据库查询没有优化
   - 建议：添加索引和分页
   - 预期改进：API 响应时间减少 50%

我会为你实现这些优化...
```

### 示例 5：代码审查

**场景**：想让 OpenCode 审查你的代码

**命令**：
```
> 请审查 src/components/UserForm.jsx 这个组件
> 检查代码质量、性能和安全性
```

**OpenCode 审查**：
```
代码审查结果：

✓ 优点：
- 组件结构清晰
- 使用了 React Hooks
- 有基本的表单验证

⚠️ 需要改进：
1. 缺少错误处理
   - 建议：添加 try-catch 块

2. 没有加载状态
   - 建议：添加 loading 状态提示用户

3. 密码字段没有加密
   - 建议：使用 HTTPS 和密码加密

4. 没有单元测试
   - 建议：添加 Jest 测试

我会为你改进这些问题...
```

## 💡 常见问题

### Q1：OpenCode 支持哪些编程语言？

**A**：OpenCode 支持所有主流编程语言，包括：
- JavaScript/TypeScript
- Python
- Java
- C++
- Go
- Rust
- 等等

### Q2：OpenCode 可以处理多大的项目？

**A**：OpenCode 可以处理任何大小的项目，但建议：
- 小项目（<1000 行）：直接使用
- 中等项目（1000-10000 行）：分步骤使用
- 大项目（>10000 行）：分模块使用

### Q3：OpenCode 生成的代码质量怎么样？

**A**：质量取决于：
- 你的需求描述清晰度
- 你选择的 AI 模型
- 你的反馈和迭代

### Q4：OpenCode 可以用于生产环境吗？

**A**：可以，但需要：
- 充分的测试
- 代码审查
- 性能优化
- 安全检查

### Q5：OpenCode 的成本是多少？

**A**：
- OpenCode 本身：免费（开源）
- AI 模型：取决于你选择的提供商
- OpenCode Zen：官方推荐的模型套餐（付费）
