# 2.1 AI 编程工具链总览

> 学习状态：已整理
> 所属阶段：第2阶段 工具、文件与版本管理
> 内容说明：重写旧 2.1 与 2.2，改为通用 AI IDE 语境。

## 本节目标

建立以 Codex 为中心的实际工作环境认知。

## 课程摘要

旧版工具课过于依赖单一 IDE，本课改为面向 Codex、编辑器、浏览器和 GitHub 的通用工具链。

## 核心要点

- Codex 负责理解仓库、分析问题、修改文件和给出执行建议。
- 编辑器负责查看文件、对比改动和本地调试，浏览器负责预览页面。
- Git 与 GitHub 负责版本管理、回滚和部署协作。

## 本节练习

列出你现在打开这个仓库时会同时用到的 3 个工具，并写出各自职责。

## 学完后你应该能做到

- 用自己的话解释本节核心概念
- 把本节内容放回当前项目里判断它的实际作用
- 在后续页面或项目中继续复用这里的方法
