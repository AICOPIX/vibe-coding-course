# 1.4 前端、后端、数据库、API、框架一张图

> 学习状态：已整理
> 所属阶段：第1阶段 认识 AI 编程与项目全景
> 内容说明：合并旧 1.5 到 1.10。

## 本节目标

把常见技术名词放到同一张认知地图里。

## 课程摘要

这一课把常见概念统一放回项目流程里，帮助你形成整体理解。

## 核心要点

- 前端负责用户看到和操作的界面。
- 后端负责业务逻辑和数据处理，数据库负责长期保存信息。
- 框架是提高开发效率的工具箱，API 负责系统之间交换数据。

## 本节练习

用“作品集网站”这个例子，分别写出前端、后端、数据库和 API 的角色。

## 学完后你应该能做到

- 用自己的话解释本节核心概念
- 把本节内容放回当前项目里判断它的实际作用
- 在后续页面或项目中继续复用这里的方法
