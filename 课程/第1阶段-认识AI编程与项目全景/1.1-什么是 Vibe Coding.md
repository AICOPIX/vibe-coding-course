# 1.1 什么是 Vibe Coding

> 学习状态：已整理
> 所属阶段：第1阶段 认识 AI 编程与项目全景
> 内容说明：基于旧 1.1 内容重写并保留核心定义。

## 本节目标

理解 Vibe Coding 的核心定义、角色分工和学习价值。

## 课程摘要

这节课建立整个项目的起点：你负责提出目标和判断结果，AI 负责生成实现。

## 核心要点

- Vibe Coding 不是放弃思考，而是把编码执行交给 AI。
- 零基础学习者的关键能力不是背语法，而是说清需求、看懂结果、持续修正。
- 判断标准始终是结果是否解决问题，而不是代码像不像工程师。

## 本节练习

写下一个你最想做的小工具，用一句话描述目标用户、核心功能和理想效果。

## 学完后你应该能做到

- 用自己的话解释本节核心概念
- 把本节内容放回当前项目里判断它的实际作用
- 在后续页面或项目中继续复用这里的方法
