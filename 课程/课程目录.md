# Vibe Coding 零基础重构版课程目录（重构中）

> 面向：完全小白、非科班、设计师视角、想用 AI 做项目的人
>
> 主线目标：先学会做出网页项目，再理解更复杂的扩展能力
>
> 核心课程：8 个阶段，35 节核心课
>
> 当前状态：第 1 到第 2 阶段已完成增强版正文；第 3 到第 8 阶段将按这份目录继续同步重构。

## 第1阶段：先看懂 AI 编程到底在做什么（4节）

这一阶段不要求你会写代码，只要求你先建立正确认知。你要先知道：你不是在“硬学编程”，而是在学“怎么和 AI 一起做项目”。

- [x] 1.1 什么是 AI 编程
- [x] 1.2 AI 编程有哪几种做法
- [x] 1.3 一个项目从想法到成品会经历什么
- [x] 1.4 前端、后端、数据库、API、框架，一次看懂

## 第2阶段：先学会和 AI 说清楚，再学更复杂的技术（5节）

很多人不是败在技术，而是败在不会表达。这一阶段先把“你怎么和 AI 协作”打好基础。

- [x] 2.1 AI 编程工具长什么样
- [x] 2.2 怎么把你的想法说清楚
- [x] 2.3 怎么把一个大任务拆成小任务
- [x] 2.4 怎么继续追问、修改和迭代
- [x] 2.5 上下文是什么，为什么 AI 有时会“忘”

## 第3阶段：开始接触项目文件、终端和版本管理（4节）

这一阶段才开始碰你以前容易怕的部分。因为前面你已经知道项目是怎么回事了，现在再学这些会顺很多。

- [ ] 3.1 项目文件、文件夹和常见文件类型
- [ ] 3.2 终端是什么，最常用命令有哪些
- [ ] 3.3 Git 是什么，为什么它像“项目存档”
- [ ] 3.4 GitHub 仓库、上传、协作和发布准备

## 第4阶段：网页项目到底是怎么组成的（6节）

这阶段开始真正进入 Web 基础，但会按小白逻辑排。不是先讲抽象定义，而是先讲“项目里为什么会遇到它”。

- [ ] 4.1 网页由什么组成：HTML、CSS、JavaScript
- [ ] 4.2 页面和数据怎么联系起来
- [ ] 4.3 JSON、配置文件、环境变量，为什么项目总会碰到它们
- [ ] 4.4 框架是什么：React、Vue、Next.js 到底差在哪
- [ ] 4.5 样式方案、组件库和 Tailwind CSS
- [ ] 4.6 数据怎么“存起来”：本地存储、数据库、持久化

## 第5阶段：外部能力、接口和那些看起来很新的词（4节）

这一阶段专门处理“新概念焦虑”。会把这些词放在你已经有前置理解的时候再讲。

- [ ] 5.1 API 是什么：让你的项目去用别人的能力
- [ ] 5.2 SDK、版本号、TypeScript，先懂到什么程度就够
- [ ] 5.3 Webhook 和自动化：为什么系统可以自己通知系统
- [ ] 5.4 MCP、Skills、智能体，这些新词到底是什么

## 第6阶段：报错、测试和安全边界（4节）

到了这里，你已经能开始做项目了。接下来最重要的不是“再加新概念”，而是学会怎么判断项目有没有做好。

- [ ] 6.1 浏览器开发者工具和常见报错阅读
- [ ] 6.2 怎么判断问题到底出在哪
- [ ] 6.3 测试、验收和回归，到底是在测什么
- [ ] 6.4 API 密钥安全、项目边界和什么不能乱给 AI

## 第7阶段：开始做真正的网页项目（4节）

这一阶段是主线实战。你不再只是在理解概念，而是开始把前面学过的东西真正串起来。

- [ ] 7.1 做网页前先做什么：信息结构、页面拆解和网格草图
- [ ] 7.2 实战：个人介绍页
- [ ] 7.3 实战：作品集网站
- [ ] 7.4 实战：把课程内容整理成静态课程站

## 第8阶段：把项目发布出去，并知道后面还能怎么走（4节）

这一阶段是项目闭环阶段。你要知道：不是做完页面就结束，而是要能发布、维护、继续扩展。

- [ ] 8.1 GitHub Pages 和常见部署平台
- [ ] 8.2 域名、DNS、CDN 和上线基础
- [ ] 8.3 设计稿转代码、UI 模板和页面改造
- [ ] 8.4 项目维护、功能扩展和下一步学习方向
