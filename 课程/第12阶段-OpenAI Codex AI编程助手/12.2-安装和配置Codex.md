## ⚙️ 第四步：配置 Codex

### 4.1 获取 API 密钥

#### 方式 1：使用 ChatGPT 账户（推荐）

```bash
codex login
```

这会：
1. 打开浏览器
2. 让你登录 ChatGPT
3. 自动配置 API 密钥

#### 方式 2：使用 OpenAI API 密钥

如果你有 OpenAI API 账户：

1. 访问：https://platform.openai.com/api/keys
2. 创建新的 API 密钥
3. 复制密钥

然后设置环境变量：

```bash
# macOS / Linux
export OPENAI_API_KEY="your-api-key"

# Windows (PowerShell)
$env:OPENAI_API_KEY="your-api-key"

# Windows (CMD)
set OPENAI_API_KEY=your-api-key
```

### 4.2 配置文件

Codex 会在以下位置查找配置文件：

| 位置 | 说明 |
|------|------|
| `~/.codex/config.json` | 全局配置 |
| `.codex/config.json` | 项目配置 |
| 环境变量 | 运行时配置 |

**创建全局配置文件**：

```bash
mkdir -p ~/.codex
```

创建文件：`~/.codex/config.json`

```json
{
  "model": "gpt-5-codex",
  "temperature": 0.7,
  "max_tokens": 4096,
  "timeout": 300
}
```

**配置说明**：

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `model` | 使用的模型 | gpt-5-codex |
| `temperature` | 创意程度（0-1） | 0.7 |
| `max_tokens` | 最大输出长度 | 4096 |
| `timeout` | 超时时间（秒） | 300 |

### 4.3 创建项目配置

在你的项目根目录创建：`.codex/config.json`

```json
{
  "model": "gpt-5-codex",
  "project_name": "my-project",
  "language": "javascript",
  "framework": "react"
}
```

## 📍 配置优先级

Codex 会按以下优先级加载配置：

1. **环境变量** - 最高优先级
2. **项目配置** - `.codex/config.json`
3. **全局配置** - `~/.codex/config.json`
4. **默认配置** - 最低优先级

**例子**：

如果你设置了环境变量 `OPENAI_API_KEY`，它会覆盖配置文件中的设置。

## 🐛 常见问题

### Q1：我在 Windows 上安装失败了怎么办？

**A**：
1. 确保你已安装 Node.js 18+
2. 打开 PowerShell（以管理员身份）
3. 运行：`npm install -g @openai/codex`
4. 如果还是失败，尝试使用 Python 版本

### Q2：我的 API 密钥在哪里？

**A**：
- 如果使用 ChatGPT 账户：运行 `codex login`
- 如果使用 OpenAI API：https://platform.openai.com/api/keys

### Q3：我可以离线使用 Codex 吗？

**A**：不行。Codex 是云端服务，需要网络连接。

### Q4：Codex 的价格是多少？

**A**：
- ChatGPT Plus：$20/月
- ChatGPT Pro：$200/月
- OpenAI API：按使用量计费

### Q5：我可以同时使用多个 API 密钥吗？

**A**：可以。在项目配置中指定不同的密钥。
