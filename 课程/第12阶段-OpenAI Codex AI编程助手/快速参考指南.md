## 📝 提示和技巧

### 技巧 1：给出清晰的需求

**不好**：
```
创建一个组件
```

**好**：
```
创建一个 React 组件，显示用户列表。
需要：
- 从 API 获取数据
- 显示用户名和邮箱
- 支持删除用户
- 使用 Tailwind CSS
```

### 技巧 2：提供上下文

```
我在做一个 React 应用，使用 TypeScript。
请创建...
```

### 技巧 3：提供示例

```
输入：2025-01-17
输出：2025年1月17日
```

### 技巧 4：分步骤进行

```
第一步：创建...
第二步：添加...
第三步：优化...
```

### 技巧 5：给出反馈

```
不对，应该是...
请修改。
```

## 🔑 API 密钥

### 获取 API 密钥

**方式 1：使用 ChatGPT 账户**
```bash
codex login
```

**方式 2：使用 OpenAI API**
1. 访问 https://platform.openai.com/api/keys
2. 创建新密钥
3. 复制密钥

### 设置环境变量

```bash
# macOS / Linux
export OPENAI_API_KEY="your-api-key"

# Windows (PowerShell)
$env:OPENAI_API_KEY="your-api-key"

# Windows (CMD)
set OPENAI_API_KEY=your-api-key
```

## 💡 常见问题

### Q1：Codex 是免费的吗？

**A**：
- ChatGPT Plus：$20/月（包含 Codex）
- ChatGPT Pro：$200/月（包含 Codex）
- OpenAI API：按使用量计费

### Q2：Codex 支持哪些编程语言？

**A**：
- JavaScript / TypeScript
- Python
- Java
- C++
- C#
- Go
- Ruby
- PHP
- Swift
- Kotlin
- 等等

### Q3：Codex 可以离线使用吗？

**A**：不行。Codex 是云端服务，需要网络连接。

### Q4：我的代码会被保存吗？

**A**：
- ChatGPT 网页版：会保存对话历史
- CLI 版本：不会保存代码
- 你可以删除对话历史

### Q5：Codex 可以访问我的文件吗？

**A**：不行。Codex 只能看到你粘贴的代码。

## 🔧 故障排除

### 问题 1：API 密钥不工作

**解决**：
1. 检查密钥是否正确
2. 检查环境变量是否设置
3. 重新生成密钥

### 问题 2：Codex 生成的代码有 Bug

**解决**：
1. 告诉 Codex 具体问题
2. 提供错误信息
3. 要求修复

### 问题 3：网络连接问题

**解决**：
1. 检查网络连接
2. 检查防火墙设置
3. 尝试使用 VPN

### 问题 4：超时错误

**解决**：
1. 简化需求
2. 分步骤进行
3. 检查网络速度
