# 第12阶段：OpenAI Codex AI 编程助手

## 📌 阶段概述

欢迎来到第12阶段！在这个阶段，你将学习 **OpenAI Codex**，一个专业级的 AI 编程助手。

与第11阶段的 OpenCode 不同，Codex 是 OpenAI 官方推出的产品，具有企业级功能，包括代码审查、多步骤任务自动化、Slack 集成等。

---

1. ✅ 理解 Codex 是什么以及它能做什么

3. ✅ 使用 Codex 编写代码和修复 Bug
4. ✅ 使用 Codex 进行代码审查和性能优化
5. ✅ 使用 Codex 完成复杂的多步骤任务
6. ✅ 用 Codex 完成真实项目

## 💡 关键概念

### Codex 的三种使用方式

| 方式 | 优点 | 适合场景 |
|------|------|---------|
| **ChatGPT 网页版** | 最简单，不需要安装 | 快速尝试、学习 |
| **CLI 命令行** | 快速，集成到开发流程 | 日常开发、自动化 |
| **IDE 扩展** | 不用切换窗口，工作流顺畅 | 专业开发 |

### Codex 的核心功能

1. **编写代码** - 根据需求生成代码
2. **修复 Bug** - 找出并修复问题
3. **代码审查** - 自动审查代码质量
4. **多步骤任务** - 自动完成复杂任务
5. **性能优化** - 优化代码性能
